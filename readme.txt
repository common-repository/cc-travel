== Description ==

<p>CC Travel is a free plugin for WordPress. If you are owning a travel website, just try CC Travel to add destinations, tours and initerary, departure date, local maps and much more information that you need. You totally can add new taxonomies or custom fields to show as much content as you want.</p>
<strong>#Let_see_how _CC _Travel_changes_your_website!</strong>
<h4>Live Demo</h4>
<p>Please click here for <a href="http://demo.chuyencode.com/tours/">CC Travel Demo</a></p>
<strong>Core features:</strong>
<ul>
<li>100% responsive.</li>
<li>Show tours in the grid or list layout.</li>
<li>Easy te set layout for the taxonomy page or archive page at CC Travel global option</li>
<li>Include Destination and Travel Style as the default taxonomies.</li>
<li>Add as many new taxonomies and custom fields as you want.</li>
<li>Convenient booking function to help users easier to book a tour.</li>
<li>Attractive booking page.</li>
<li>Easy to change the currency and currency position</li>
<li>Flexible tour page layout to adapt to your current website.</li>
<li>Show initerary for tours.</li>
<li>Show the departure date for tours. You can choose “All Days”, or choose some typical days.</li>
<li>Tons of options so that you can add full information (description, destination, travel style, itinerary, departure, poster, banner, gallery, and much more) that you need for the tours.</li>
<li>Support location maps.</li>
<li>Options to show highlights information for tours as well as the destination.</li>
<li>Especially, CC Travel includes 2 shortcodes: Tour Grid and Tour List shortcode. It will help you add the tours to any page you want.</li>
</ul>

<strong>If you want to see more detail, just find the information below. CC Travel is totally free now. </strong>
